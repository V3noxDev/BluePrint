# MineSuite — Blueprint Extension

Extensão Blueprint para Pterodactyl com editor de `server.properties` e instalador/removedor de addons Minecraft Java.

Tudo fica em [`minesuite/`](./minesuite/). Veja o [README da extensão](./minesuite/README.md) para instalação e uso.

```bash
# Empacotar
cd minesuite && ./scripts/pack.sh
```
