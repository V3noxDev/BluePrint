# BluePrint — MineControl

Pterodactyl **Blueprint** extension for Minecraft Java servers.

## MineControl

Polished panel UI to:

- **Install / remove addons** (Modrinth search, multi-select delete, enable/disable)
- **Edit `server.properties`** with categorized controls and automatic backups

### Quick install

```bash
cp minecontrol.blueprint /var/www/pterodactyl/
cd /var/www/pterodactyl
blueprint -install minecontrol
```

Source and docs: [`minecontrol/README.md`](minecontrol/README.md)
